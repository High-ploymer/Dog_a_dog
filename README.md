# 狗了个狗 （C++ & EasyX）

## 项目简介/Project Introduction

一款基于C++和EasyX开和羊了个羊改编的热门三消小游戏。

## 项目特点/Project Features

开发难度低：项目总体难度中等，适合初学者参考学习。

图形界面：引入EasyX图形库，使得游戏界面更加美观。

关卡式设计：游戏提供多个关卡，在关卡中方块数量种类的增加的同时，时间的限制也在更新，考验用户的手速和眼力哦

参数可调：关卡难度、图案数量定义于代码开头部分，可根据需要进行调整。

## 游戏规则/Game Rules

游戏开始时，地图中会随机生成一定数量的图案。关卡数越大，图案数量和种类越多。

初始的时间限制为40s 随着关卡的增加 时间仅增加5s 若选择难度增加模式 第一关初始时间为30s。

玩家通过点击画面内的图案，将其转移到画面下方的卡槽中，卡槽的存放上限为7。

如果卡槽中相同的图案达到3个则会消除。当地图中的图案全部消除后，通过该关卡，难度升级。

如果卡槽中的图案达到上限，则游戏结束。

参数：最高关卡可自行设置，每关图案数量 = 20 + (int)(10 * sqrt(Level - 1))
